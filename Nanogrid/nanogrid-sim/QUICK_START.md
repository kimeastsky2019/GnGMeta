# 🚀 빠른 시작 가이드

## 1️⃣ 파일 다운로드
`nanogrid-digital-twin-v2.zip` 파일을 다운로드하고 압축 해제

## 2️⃣ 설치
```bash
cd nanogrid-digital-twin-v2
npm install
```

## 3️⃣ 개발 서버 실행
```bash
npm run dev
```

브라우저에서 http://localhost:5173 접속

## 4️⃣ 기능 확인

### 대시보드
- 홈 화면에서 실시간 데이터 확인
- API가 아직 구현되지 않았다면 모의 데이터로 표시
- 30초마다 자동 업데이트

### 시뮬레이터  
- 상단 메뉴 "시뮬레이터" 클릭
- "API에서 실시간 데이터 가져오기" 버튼 테스트
- 또는 "합성 데이터 사용"으로 오프라인 테스트
- CSV 파일 업로드도 가능

## 5️⃣ API 연동 (백엔드 준비 시)

`src/lib/api.js` 파일 확인:
```javascript
const API_BASE = 'https://agent.gngmeta.com'
```

백엔드 URL이 다르면 수정하세요.

## 📝 주요 기능

### ✨ 대시보드
- 실시간 에너지 모니터링
- 인포그래픽 카드
- 24시간 시계열 차트
- 자급률, 비용 절감, CO₂ 저감

### 🎯 시뮬레이터
- 3가지 시나리오 자동 비교
- CSV 업로드 지원
- API 실시간 데이터 연동
- 결과 저장 및 조회

## 🐛 문제 해결

### API 연결 안 됨
→ 정상입니다! 백엔드가 없으면 모의 데이터로 작동합니다.

### 차트가 안 보임
→ 브라우저 콘솔(F12)에서 에러 확인

### 스타일이 깨짐
→ `npm install` 다시 실행
