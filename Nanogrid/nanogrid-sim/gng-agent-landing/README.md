# Gng AI Cloud Voucher Agent Service Landing Page

This is the landing page for the Gng AI Cloud Voucher Agent Service, built with Next.js, Tailwind CSS, and GSAP.

## Getting Started

Since the project was initialized manually, you need to install the dependencies first.

1.  Navigate to the project directory:
    ```bash
    cd gng-agent-landing
    ```

2.  Install dependencies:
    ```bash
    npm install
    # or
    yarn install
    # or
    pnpm install
    ```

3.  Run the development server:
    ```bash
    npm run dev
    # or
    yarn dev
    # or
    pnpm dev
    ```

4.  Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

-   `src/app`: App Router pages and layout.
-   `src/components/sections`: Landing page sections (Hero, Problem, Solution, etc.).
-   `src/lib`: Utility functions.

## Features

-   **Next.js 14 App Router**
-   **Tailwind CSS** for styling.
-   **GSAP** for scroll animations (Solution section).
-   **Lucide React** for icons.
